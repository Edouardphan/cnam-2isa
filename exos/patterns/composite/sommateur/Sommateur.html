Exercice : Sommateur

Code source de l'énoncé

Compléter Sommateur.java pour obtenir un composite dont :
  • les feuilles ont une valeur (de type double).
  • La valeur d'un noeud composite est la somme des valeurs de ses enfants.
  • Chaque noeud a une méthode somme() qui renvoie la somme.

Dans la fonction main() :
  • Remplissez le composite avec des valeurs de tests.
  • Affichez la somme totale.
  • Affichez les sommes des enfants directs du noeud racine, avec une boucle "foreach"
