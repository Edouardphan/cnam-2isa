Exercice : GUI

Utilisation du pattern observateur en simulant les interactions pouvant exister dans un interface graphique (GUI, Graphical User Interface).
Code source de l'énoncé Une application graphique permet à l'utilisateur d'effectuer trois sortes d'actions différentes : new, save, exit.
Ces actions peuvent être commandées de deux manières différentes : par le biais de menus et par le biais de boutons.
