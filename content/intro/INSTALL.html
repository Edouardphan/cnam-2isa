INSTALL - Prérequis

La prise en main de l'environnement de travail est donnée en Bash sous Linux (Ubuntu) mais utilisez les moyens que vous voulez, où vous vous sentez le plus pour l'aise de travailler, par exemple Powershell ou VSCode ou Eclipse.
L'essentiel est de travailler avec git et de pouvoir écrire et exécuter des petits programmes dans différents langages.

Git

Installer git
Doc brouillon dans la section git 
sudo apt install git
Cloner le répertoire du cours
git clone https://github.com/tig12/cnam-2isa.git
Crée un dossier cnam-2isa dans le répertoire courant.
Tester git
cd cnam-2isa/
git remote -v
Affiche 
origin	https://github.com/tig12/cnam-2isa.git (fetch)
origin	https://github.com/tig12/cnam-2isa.git (push)
Récupérer la dernière version du cours : 
git pull
